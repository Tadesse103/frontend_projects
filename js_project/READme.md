# Task Management Website

## Overview

This is a simple task management website that allows users to create, manage, and track their tasks efficiently. The application is designed to help users stay organized and improve productivity.

## Features
 
- **Task Creation**: Users can create new tasks with titles, descriptions, and due dates.
- **Task Management**: Users can edit, delete, and mark tasks as completed.
- **Filter and Sort**: Users can filter tasks by status and sort them by due date.
- **Responsive Design**: The website is mobile-friendly and works on various devices.
